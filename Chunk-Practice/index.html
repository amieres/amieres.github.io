How to use Recital Practice Chunks

What it does

Cuts your sheet music into short phrases ("chunks") and serves them in random order. Each chunk shows a window onto the original score plus the first note of the next phrase, so the chunks chain together in your memory. Drill the hard parts without mindless start-from-the-top runs.

Adding a piece
  1. Drop your sheet-music images (JPG / PNG) into the same folder as index.html. Each page of the score is one image.
  2. Click + New piece in the top toolbar.
  3. Enter a piece name, click Add page (pick image) for each image (the file picker reads dimensions automatically and uses the filename as a relative path).
  4. Click Save piece.
Adding chunks

Two ways:

  • Manually: filter to the piece, click + New chunk, drag the rectangle over the measures you want, fill in title / description / connector, and save. Use + Add rect on this page for chunks that span line breaks.
  • With Claude: open the piece in Edit piece and click ✨ Generate chunks with Claude. You will need an Anthropic API key (set in Settings). Cost is roughly 5–20¢ per page; the result is a 70%-there starting layout you can tweak.
Editing a chunk window

Click Edit this chunk while a chunk is showing. You see the relevant page(s) at full size with yellow rectangles. Drag the body to move; drag corners or edges to resize. The active rectangle is highlighted; click another to switch focus.

Each page also has a red “sig end” ruler — drag it horizontally to mark where the line clef + key + time signature end. The tool auto-prepends a clef strip onto any chunk whose rectangle starts past the sig-end x.

Practising

Filter to the piece (or All pieces), click Random chunk. Play the chunk through the bridge note. Click Got it — next to mark it practised and roll a new one, or Skip / re-roll to keep it in the pool but try a different one. Reset cycle brings everything back.

Backup & share

Use Export to download a JSON of your pieces, pages, chunks, and page settings. Import reads it back. Keep the JSON next to your image files to make a setup portable.

If a future version ships better defaults, a banner appears at the top on next load. Settings → Reset to bundled defaults wipes your edits (export first if you want a backup) and re-seeds from the file defaults.
